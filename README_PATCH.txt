PATCH BLINDADO — Jogo (turnos + HUD + arena + visuais Orion/Solaris)

Como aplicar:
1) Copie a pasta 'Jogo/' para a raiz do seu projeto (onde existe 'Assets/').
2) No PowerShell: Set-ExecutionPolicy -Scope Process Bypass -Force; .\preflight.ps1
3) Rode .\clean_open_6000.cmd (ou abra pelo Hub na 6000.2.9f1).
4) Dê Play: a arena e os personagens aparecem em runtime; HUD no canto inferior.
