
# Clean‑Room RPG — Projeto Completo (v1→v9)
Abra no **Unity 2022.3 LTS**. Cena inicial: `Assets/Scenes/Main.unity` (boot automático).
Controles: F2 Seleção, F3 Gacha, 1/2/3 habilidades, Tab alvo, Espaço passar, R reset.

## v10 extra
- HUD melhorada, visuais 3D com diferenciadores por herói (martelo, halo, escudo, orbes).
- Arena PvP local (F4) com **pick/ban**, 3x3, botão Auto Enemy.

## v11 extra
- **Art Pack integrado** (Icons/Matcaps) para retratos e ícones de habilidades.
- **Arena PvP expandida**: 2 bans por lado, picks 5×5, timer por fase (20s), alternância de first-pick (botão).

## v12 extra
- **Shader Pack**: `Shaders/MatcapOutline.shader` (matcap + outline), aplicado em personagens.
- **Cena**: `SceneBuilder` cria starfield, ruínas e piso de pedras.
- **Replay**: `ReplaySystem` + `ReplayUI` (F5) — grava ações e reproduz em 0.25× para ver combos em câmera lenta.
